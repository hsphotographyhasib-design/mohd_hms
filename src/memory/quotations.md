# FacilityPro — Quotations Feature

## Overview
The quotations system manages price quotes for customers. Quotations track line items, taxes, discounts, and can be converted to work orders or invoices.

## Data Model (`Quotation` in Prisma schema)

### Key Fields
| Field | Type | Description |
|-------|------|-------------|
| `id` | String (cuid) | Primary key |
| `tenantId` | String | Tenant scope |
| `customerId` | String | FK to Customer |
| `complaintId` | String? | Optional link to Complaint |
| `quotationNo` | String? | e.g. QTN/SMSB/01/4529 |
| `title` | String | Quotation title |
| `description` | String? | Description |
| `referenceNo` | String? | Internal reference |
| `projectName` | String? | Project name |
| `site` | String? | Site location |
| `preparedBy` | String? | FK to User |
| `items` | String | JSON array of line items |
| `terms` | String? | JSON array of terms & conditions |
| `currency` | String | Default: "BND" |
| `subtotal` | Float | Items total |
| `taxRate` | Float | Tax percentage |
| `tax` | Float | Tax amount |
| `discount` | Float | Discount amount |
| `shipping` | Float | Shipping cost |
| `total` | Float | Grand total |
| `status` | String | 11 possible statuses |
| `validUntil` | DateTime? | Expiration date |
| `pdfUrl` | String? | Generated PDF URL |

### Relations
- `tenant` → Tenant
- `customer` → Customer
- `invoices` → Invoice[] (quotations can generate multiple invoices)

## Quotation Statuses (11)
| Status | Description |
|--------|-------------|
| `DRAFT` | Initial draft |
| `REVIEW` | Under review |
| `APPROVED` | Internally approved |
| `SENT` | Sent to customer |
| `ACCEPTED` | Customer accepted |
| `REJECTED` | Customer rejected |
| `EXPIRED` | Past validUntil date |
| `CONVERTED_WO` | Converted to work order |
| `CONVERTED_INVOICE` | Converted to invoice |
| `PAID` | Fully paid |
| `CLOSED` | Closed/cancelled |

## Line Item Structure (JSON in `items` field)
```json
[
  {
    "id": "temp_xxx",
    "title": "HVAC Compressor Replacement",
    "description": "Brand new compressor unit",
    "unit": "unit",
    "quantity": 1,
    "rate": 2500.00,
    "amount": 2500.00,
    "category": "Parts",
    "warranty": "12 months"
  }
]
```

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/quotations` | List (supports `?stats=true` for aggregate stats) |
| POST | `/api/quotations` | Create quotation |
| GET | `/api/quotations/[id]` | Get single quotation |
| PUT | `/api/quotations/[id]` | Update quotation |
| DELETE | `/api/quotations/[id]` | Delete quotation |
| POST | `/api/quotations/create` | Create with full payload |
| GET | `/api/quotations/next-number` | Generate next quotation number |
| PUT | `/api/quotations/[id]/status` | Update quotation status |
| GET | `/api/quotations/item-suggestions` | Search items for line items |

### Stats Mode (`?stats=true`)
Returns aggregate data: total, totalValue, draft, pendingAction, accepted, paid.

### Totals Computation
```ts
computeTotals(items, taxRate, discount, shipping) → { subtotal, tax, discount, shipping, total }
```

## Quotation Number Format
`QTN/SMSB/01/{YEAR}/{SEQ}` — generated by `getNextQuotationNumber()` endpoint.
