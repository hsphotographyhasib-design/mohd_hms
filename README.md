# 📖 Overview

MOHD.HMS Enterprise is an enterprise-grade AI-powered Smart Facility Maintenance Management System designed to help organizations manage assets, maintenance operations, technicians, projects, customers, inventory, quotations, work orders, inspections, preventive maintenance, and financial operations from a single platform.

The system combines the capabilities of:

* Enterprise Asset Management (EAM)
* Computerized Maintenance Management System (CMMS)
* Computer Aided Facility Management (CAFM)
* Project Management
* Customer Relationship Management (CRM)
* AI Assistant
* Business Intelligence Dashboard

# ⚡ Installation

```bash
git clone https://github.com/hsphotographyhasib-design/mohd_hms.git

cd mohd_hms

npm install

npm run dev
```

Open:

```
http://localhost:3000
```

---

# ⚙️ Environment Variables

Create:

```
.env.local
```

Example:

```env
DATABASE_URL=

NEXTAUTH_SECRET=

NEXTAUTH_URL=http://localhost:3000

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

FACEBOOK_CLIENT_ID=

FACEBOOK_CLIENT_SECRET=

WHATSAPP_API_KEY=
```
